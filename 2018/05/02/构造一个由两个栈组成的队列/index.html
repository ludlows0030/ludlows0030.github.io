构造一个由两个栈组成的队列

如何构造一个由两个栈组成的队列

队列所需的操作

  • enqueue ：将一个元素加入到队列的末尾
  • dequeue ： 将一个元素从队列的顶部移除

栈的操作

  • pop : 将一个元素从栈顶移除
  • push ：将一个元素加入到栈顶

思路

  • 可以将队列分为两部分 inbox outbox(两个栈)
  • 当outbox为空的时候，将inbox的元素逐个pop并push到outbox中，此时outbox的栈顶为最先加入到inbox的元素

代码

public static class QueueWithTwoStakcs{
  private Stack<Item> inbox = new Stack<Item>;
  private Stack<Item> outbox = new Stack<Item>;

  public void enqueue(Item item){
    inbox.push(item);
  }

  public Item dequeue(){
    if(outbox.isEmpty()){
      while(!inbox.isEmpty){
        outbox.push(inbox.pop());
      }
    }
    return outbox.pop();
  }
}
