排序

实现排序的模板

public class Example{
  public static void sort(Comparable[] a){
    //各算法具体实现
  }

  //比较两个实现Comparable接口的数据类型的大小
  private static boolean less(Comparable v, Comparable w){
    return v.compareTo(w)<0;
  }

  //将数组中的两个值进行交换
  private static void exch(Comparable[] a,int i ,int j){
    Comparable t = a[i];
    a[i] = a[j];
    a[j] = t;
  }

  //打印数组
  private static void show(Comparable[] a){
    for(int i = 0;i<a.length;i++){
        StdOut.print(a[i] + " ");
    }
    StdOut.println();
  }

  //数组是否有序
  public static boolean isSorted(Comparable[] a){
    for(int i = 1;i<a.length;i++){
        if(less(a[i],a[i-1])) return false;
    }
    return true;
  }

  public static void main(String[] args){
    String[] a = StdIn.readStrings();
    sort(a);
    assert isSorted(a);
    show(a);
  }
}

选择排序

依次在剩余的对象中挑选出最小的放在数组的左边，每次挑选时数组的左边为有序，数组的右边为无序

public class Selection{
  public static void sort(Comparable[] a){
    int N = a.length;
    for(int i = 0;i<N;i++){
        int min = i;
        for(int j = i+1;j<N;j++){
            if(less(a[j],a[min])) min = j;
        }
        exch(a,i,min);
    }
  }

}

插入排序

将每一个对象与其前一个对象进行比较，如果比前一个对象小，则两者交换位置，直至其比其前一个对象大

public class Insertion{
    public static void sort(Comparable[] a){
        int N = a.length;
        for(int i = 1;i<N;i++){
            for(int j = i;j>0&&less(a[j],a[j-1]);j--){
                exch(a,j,j-1);
            }
        }
    }

希尔排序

以插入排序为基础，将每一个对象与其前h个对象进行比较（h为一个递增序列通常为3h+1,比较进行一轮后h/3），当h=1时相当于进行插入排序。原因是插入排序对h有序（对象间隔h个有序）的队列排序时速度较快。

public class Shell{
    public static void sort(Comparable[] a){
        int N = a.length;
        int h = 1;
        while(h<N/3) h = 3*h+1;//1,4,13,40,121,364...
        while(h>=1){
            for(int i = h;i<N;i++){
                for(int j = i;j>0&&less(a[j],a[j-h]);j-=h){
                    exch(a,j,j-h);
                }
            }
            h = h/3;
        }
    }
  ...
}
