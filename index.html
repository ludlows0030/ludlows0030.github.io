二分查找树

二分查找树

本质结构

二分查找树由结点组成，结点包含的链接可以为空（null）或者指向其他结点。在二叉树中，每个结点只能有一个父结点（只有一个例外，也就是根结点，它没有父结点），而且每个结点都只有左右两个链接，分别指向自己的左子结点和右子结点。尽管链接指向的是结点，但也可以将每个链接看作指向了另一颗二叉树，而这棵树的根结点就是被指向的结点。另外每个结点还包含了一个键和一个值，键之间是有序的

定义

一棵二叉查找树（BST）是一棵二叉树，其中每个结点都含有一个Comparable的键（以及相关联的值）且每个结点的键都大于其左子树中的任意结点的键而小于右子树中的任意结点的键

每个方法的实现思路

  • 查找（get)

    • 当查找的键大于当前结点的键，则在右子树中继续查找
    • 当查找的键小于当前结点的键，则在左子树种继续查找
    • 当查找的键等于当前的结点的键，则返回当前结点的值
    • 当当前结点为null时，则返回null

  • 插入(put)

    • 与get方法类似，当整个二叉树中没有相应键时创建新的结点，如果有相应键则更新值

查找与插入的递归有些相似之处，递归调用前相当于沿着树向下走，它会将给定的键和每个结点的键相比较并根据结果向左或者向右移动到下一个结点。然后将递归调用后的代码想象成沿着树向上爬。

对于get方法，这对应着一系列的返回指令

对于put方法，意味着重置搜索路径上的每个父结点指向子结点的链接，并更新路径中的每个结点计数器（因为插入了新的结点）

  • 最小键（min）

    • 如果根结点的左链接为空，那么一棵二叉查找树的最小值就是根结点；如果左链接非空，那么树中的最小键就是左子树中的最小键

  • 向下取整（floor）

    • 如果给定的键小于二叉树中的根结点，那么向下取整的键就在二叉树中的左子树里；如果给定的键大于二叉树中的根结点，那么向下取整的键如果不在右子树中，即为树的根结点。

  • 选择（select）

    • 如果左子树种的结点数t大于k,继续在左子树中查找排名为k的键；如果t等于k,根结点就是目标；如果t小于k，那么就在右子树种查找排名为k-t-1的键

  • 排名 (rank)

    • 如果给定的键和根结点的键相等，我们返回左子树中的结点总数t；如果给定的键小于根结点，返回键在左子树中的排名；如果大于，返回在右子树中的排名+t+1

  • 删除最小键(deleteMin)

    • 不断深入根结点的左子树中直至遇见一个空链接，然后将指向该结点的链接指向该结点的右子树。

  • 删除操作（delete）

    • 将指向即将被删除的结点的链接保存为t
    • 将x随机地指向后继结点或者前驱结点min(t.right)或者max(t.left)
    • 将x的右链接指向deleteMin(t.right)
    • 将x的左链接指向t.left

代码

public class BST<Key extends Comparable<Key>, Value>
{
    private Node root;

    private class Node
    {
        private Key key;
        private Value val;
        private Node left,right;
        private int N;

        public Node(Key key,Value val,int N)
        {
            this.key = key;
            this.val = val;
            this.N = N;
        }
    }

    public int size()
    {
        return size(root);
    }

    private int size(Node x)
    {
        if(x == null) return 0;
        else return x.N;
    }

    public Value get(Key key)
    {
        return get(root,key);
    }

    private Value get(Node x,Key key)
    {
        if(x==null) return null;//树没有相应的key
        //二分法
        int cmp = key.compareTo(x.key);
        if(cmp>0) get(x.right,key);//大，找右边
        else if(cmp<0) get(x.left,key);//小，找左边
        else return x.val;
    }

    public void put(Key key,Value val)
    {
        root = put(root,key,val);
    }

    private Node put(Node x,Key key,Value val)
    {
        if(x==null) return new Node(key,val,1);//如果当前结点为空，则创建一个新结点
        int cmp = key.compareTo(x.key);
        if(cmp>0) x.right = put(x.right,key,val);//大，找右边
        else if(cmp<0) x.left = put(x.left,key,val);//小，找左边
        else x.val = val;//相等，更新值
        x.N = size(left)+size(right)+1;//更新结点计数器
        return x;//返回自身，相当于迭代完成后，沿着路径向上重置
    }

    public Key min()
    {
        return min(root).key;
    }

    private Node min(Node x)
    {
        if(x.left == null) return x;//空返回x，已经在整个树的最左边了
        return min(x.left);//否则继续朝左边找
    }

    public Key floor(Key key)
    {
        Node x = floor(root,key);
        if(x == null)return null;
        return x.key;
    }

    private Node floor(Node x,Key key)
    {
        if(x == null) return null;//如果结点为空，返回空
        int cmp = key.compareTo(x.key);
        if(cmp==0) return x;//命中
        if(cmp<0) return floor(x.left,key);//小，找左边
        //键大于根节点，在右边找
        Node t = floor(x.right,key);
        if(t!=null) return t;//当t存在时，返回t
        else x;//不存在时，返回当前结点
    }

    public Key select(int k)
    {
        return select(root, k).key;
    }

    private Node select(Node x,int k)
    {
        if(x == null) return null;
        t = size(x.left);//左子树结点数量
        if(t>k) select(x.left,k);//t大于k ,排名为k的结点在左子树
        else if(t == k) return x;//t等于k ,排名为k的结点就是当前结点
        else select(x.right,k-t-1);//t小于k,结点在右子树，但是需要找在右子树种排名k-t-1的结点（除去左子树的结点数和根结点）
    }

    public int rank(Key key)
    {
        return rank(key,root);
    }

    private int rank(Key key,Node x)
    {
        if(x == null) return 0;
        int cmp = key.compareTo(x.key);
        if(cmp<0) return rank(key,x.left);
        else if(cmp>0) return 1+size(left)+rank(key,x.right);//根结点+左子树结点数量+结点在右子树中的排名
        else return size(left);//如果找到则返回结点左子树的结点数量，即结点在右子树中的排名
    }

    public void deleteMin()
    {
        root = deleteMin(root);
    }

    private Node deleteMin(Node x)
    {
        if(x.left == null) return x.right;//如果x.left为空，则x是最小的，返回x.right
        x.left = deleteMin(x.left);//否则继续在左子树种寻找
        x.N = size(left)+size(right)+1;//重置结点计数器
        return x;//返回当前结点 和put()中类似
    }

    public void delete(Key key)
    {
        root = delete(root,key);
    }

    private Node delete(Node x,Key key)
    {
        if(x == null) return null;
        int cmp = key.compareTo(x.key);
        if(cmp>0) x.right = delete(x.right,key);
        else if(cmp<0) x.left =delete(x.left,key);
        else
        {
            if(x.right==null) return x.left;
            if(x.left==null) return x.right;

            Node t = x;
            x = min(t.right);
            x.right = deleteMin(t.right);
            x.left = t.left;
        }
        x.N = size(left)+size(right)+1;
        return x;
    }
}

基于二分查找的有序数组实现的符号表

基于二分查找的有序数组实现的符号表

符号表应该要实现哪些操作

  • 将一个键值对插入符号表
  • 在之后能够从符号表的所有键值对中按照键直接找到相应的值

符号表的定义

  • 符号表是一种存储键值对的数据结构，支持两种操作：插入（put），即将一组新的键值对存入表中；查找（get），即根据给定的键得到相应的值

符号表的关于重复键的规则

  • 每个键只对应一个值 并不允许存在重复的键
  • 当用例代码向表中存入的键值对和表中已有的键冲突时，新的值会替代旧的值

空键的情况

  • 键不能为空，使用空键会产生一个运行时的异常

空值的情况

  • 不允许有空值
  • 用get()方法返回空来测试给定的键是否存在于符号表
  • 我们可以将空值作为put()方法的第二个参数存入表中来实现删除

删除操作

  • 延时删除 ： 将键对应的值置为空，然后在某个时候删去所有值为空的键
  • 即时删除 ： 立刻从表中删除指定的键

排名和选择

  • 检验一个新的键是否插入合适位置的基本操作
  • rank() 找出小于指定键的键的数量
  • select() 找出排名为k的键
  • 通常有 i == rank(select(i)) 以及 key == select(rank（key))

实际情况中的能处理大量初级的符号表应具备的特性

  • 混合使用查找和插入的操作
  • 大量不同的键
  • 查找操作比插入操作多得多
  • 虽然不可预测，但查找和插入操作的使用模式并非随机

有序数组的二分查找

  • 核心是rank()方法，用于定位，确定位置后，再用put()和get()方法

具体实现

public class BinarySearchST<Key extends Comparable<key>, Value>
{
    //两个数组，一个存储键，一个存储值
    private Key[] keys;
    private Value[] vals;
    private int N;

    //构造函数，构造传入参数大小的数组
    public BinarySearchST(int capacity)
    {
        keys = (Key[]) new Comparable[capacity];
        vals = (Value[]) new Comparable[capacity];
    }

    //当表内存在键时，更新值，否则插入一个新的键值对
    public void put(Key key,Value val)
    {
        if(val = null) delete(key);return;
        int i = rank(key);
        if(i<N&&keys[i].compareTo(key) == 0)
        {
            vals[i] = val;
        }


        for(int j = N;j>i;j--)
        {
            keys[j] = keys[j-1];
            vals[j] = vals[j-1];
        }
        keys[i] = key;
        vals[i] = vals;
        N++;
    }

    //表为空时 返回null，key不在表内时返回null
    public Value get(Key key)
    {
        if(isEmpty())return null;
        int i = rank(key);
        if(i<N&&keys[i].compareTo(key)==0) return vals[i];
        else return null;
    }

    //删除键值对，并把其后键值对向前移
    public Key delete(Key key)
    {
        int i = rank(key);
        if(i<N&&keys[i].compareTo(key) == 0 )
        {
            vals[i] = null;
            for(int j = i;j<N;j++)
            {
                keys[j] = keys[j+1];
                vals[j] = vals[j+1];
            }
            N--;
            return key;
        }
        else return null;
    }

    //返回表内是否存在key键的布尔值
    public boolean contains(Key key)
    {
        int i = rank(key);
        return i<N&&keys[i].compareTo(key) == 0;
    }

    //表是否为空
    public boolean isEmpty()
    {
        return size() == 0;
    }

    //表的大小
    public int size()
    {
        return N;
    }

    //表中最小键
    public Key min()
    {
        return keys[0];
    }

    //表中最大键
    public Key max()
    {
        return keys[N-1];
    }

    //表中小于等于key键的最大键
    public Key floor(Key key)
    {
        int i = rank(key);
        return keys[i-1]
    }

    //表中大于等于key键的最小键
    public Key ceiling(Key key)
    {
        int i = rank(key);
        return keys[i];
    }

    //二分查找
    public int rank(Key key,int lo,int hi)
    {
        if(hi<lo) return lo;
        int mid = lo +(hi-lo)/2;
        int cmp = key.compareTo(keys[mid]);
        if(cmp<0)
            rank(key,lo,mid-1);
        else if(cmp>0)
            rank(key,mid+1,hi);
        else
            return mid;
    }

    //二分查找 非迭代
    public int rank(Key key)
    {
        int lo = 0,hi = N-1;
        while(lo<=hi)
        {
            int mid = lo+(hi-lo)/2;
            int cmp = key.compareTo(keys[mid]);
            if(cmp<0) hi = mid-1;
            else if (cmp>0) lo = mid+1;
            else return mid;
        }
    }

    //返回表中第k-1个key
    public Key select(int k)
    {
        return keys[k];
    }

    //删除最小值
    public void deleteMin()
    {
        delete(min());
    }

    //删除最大值
    public void deleteMax()
    {
        delete(max());
    }

    //返回指定区域的大小
    public int size(Key lo,Key hi)
    {
        if(hi.compareTo(lo)<0)
            return 0;
        else if(contains(hi))
            return rank(hi)-rank(lo)+1;
        else
            return rank(hi)-rank(lo);
    }

    //返回一个队列，存储指定区间内的键
    public Iterable<Key> keys(Key lo, Key hi)
    {
        Queue<Key> q = new Queue<Key>();
        for(int i = rank(lo);i<rank(hi);i++)
        {
            q.enqueue(keys[i]);
            if(contains(hi))
                q.enqueue(keys[rank(hi)]);
            return q;
        }
    }

    public Iterable<Key> keys()
    {
        return keys(min(),max());
    }
}

优先队列

优先队列

为什么我们需要优先队列？

很多情况下，我们处理元素时，不一定要求它们全部有序，或是不一定要一次就将它们排序。很多情况下我们会收集一些元素，处理当前键值最大的元素，然后再收集更多的元素，再处理当前键值最大的元素。

例如，可能有一台能够同时运行多个应用程序的电脑。通过为每个应用程序的事件分配一个优先级，并总是处理下一个优先级最高的事件来实现的。

优先队列应该支持什么样的操作

  • 删除最大的元素
  • 插入元素

基于二叉堆数据结构实现的数据队列

API

public class MaxPQ<Key extends Comparable<Key>>
{
  MaxPQ()//创建一个优先队列
  MaxPQ(int max)//创建一个初始容量为max的优先队列
  MaxPQ(Key[] a)//用a[]中的元素创建一个优先队列

  void insert(Key v) //向优先队列中插入一个元素
  Key max() //返回最大元素
  Key delMax() //删除并返回最大元素
  boolean isEmpty() //返回队列是否为空
  int size() //返回优先队列中的元素个数
}

一个优先队列的用例

public class TopM
{
  public static main(String[] args)
  {
    int M = Integer.parseInt(args[0]);
    MinPQ<Transaction> pq = new MinPQ<Transaction>(M+1);
    while(StdIn.hasNextLine())
    {
      pq.insert(new Transaction(StdIn.readLine()));
      if(pq.size()>M)
      {
        pq.delMin();//当元素个数超出时，删除最小的以保证数组内元素为最大的M个
      }
    }

    Stack<Transaction> stack = new Stack<Transaction>();
    while(!pq.isEmpty()) stack.push(pq.delMin());//依次删除最小元素push进栈，栈内为从小到大的顺序
    for(Transaction t : stack) StdOut.println(t);
  }
}

堆的定义

堆有序： 当一棵二叉树的每个结点都大于等于它的两个子节点时，称为堆有序

从任意结点向上，我们都能得到一列非递减的元素；
从任意结点向下，我们都能得到一列非递增的元素。

二叉堆： 一组能够用堆有序的完全二叉树排列的元素，并在数组中按照层级储存（为方便计算，不是用数组的第一个位置）

在一个堆中，位置k的结点的父结点的位置为 k/2，而它的两个子结点的位置分别为2k和2k+1。

堆的算法

为实现优先队列删除最大元素和插入元素的操作，必须先实现堆有序化中，上浮(swim)和下潜(sink)的操作

private void swim(int k)
{
  while(k>1&&less(k/2,k))
  {
    exch(k/2,k);
    k/2;
  }
}

private void sink(int k)
{
  while(k*2<=N)
  {
    int j = k*2;
    if(j<N&&less(j,j+1)) j++;
    if(!less(k,j)) break;
    exch(k,j);
    k=j;
  }
}

  • 插入元素操作

    • 将新元素添加到末尾
    • 使其上浮到合适位置

  • 删除最大元素操作

    • 将根结点和最后一个结点交换
    • 令交换后的根结点下潜到合适位置

基于堆的优先队列

public class MaxPQ<Key extends Comparable<Key>>
{
    private Key[] pq;
    private int N = 0;

    public MaxPQ(int maxN)
    {
        pq = (Key[]) new Comparable[maxN+1];
    }

    public boolean Empty()
    {
        return N == 0;
    }

    public int size()
    {
        return N;
    }

    public void insert(Key v)
    {
        pq[++N] = v;//数组最后添加一个结点
        swim(v);//上浮该结点
    }

    public Key delMax()
    {
        Key max = pq[1];//根节点为最大的
        exch(1,N--);//将根节点和最后一个结点交换
        pq[N+1] = null;//交换后=null防止对象游离
        sink(1);//将交换上来的最后一个结点下沉
        return max;
    }

    private boolean less(int i, int j)
    {
        return pq[i].compareto(pq[j]) < 0;
    }

    private void exch(int i, int j)
    {
        Key temp = pq[i];
        pq[i] = pq[j];
        pq[j] = temp;
    }

    private void swim(int k)
    {
        while(k>1&&less(k/2,k))//当结点到达根节点停止循环或者父节点比当前结点大
        {
            exch(k/2,k);
            k/2;
        }
    }

    private void sink(int k)
    {
        while(k*2<=N)
        {
            int j = k*2;
            if(j<N&&less(j,j+1)) j++;
            if(!less(k,j)) break;
            exch(k,j);
            k = j;
        }
    }
}

快速paixv

快速排序

关于快速排序需要了解的

  • 快速排序为一种原地排序
  • 快速排序是一种分治的排序算法。将一个数组分成两个子数组，将两部分独立地进行排序
  • 快速排序与归并排序的不同
    • 归并排序将数组分成两个子数组分别排序，再将有序的子数组合并使得整个数组变得有序
    • 快速排序将数组分成两个子数组（以一个数k为分界，比k小的在左边，大的在右边），当子数组有序时，整个数组有序
  • 快速排序的切分策略
    • 先随意的取a[lo]为切分元素
    • 然后从数组左端开始扫描直到找到一个大于或等于它的元素
    • 再从数组右端扫描直到找到一个小于或等于它的元素
    • 将两者交换
    • 如此继续，扫描完毕时（两个扫描指针i,j相遇），将a[lo]与a[j]交换，将切分元素放到正确的位置
    • 保证a[j]左侧的元素都小于a[j]，右侧的元素都大于a[j]

具体实现

public class Quick{

  @SuppressWarnings("unchecked")
  private static boolean less(Comparable v, Comparable w){
      return v.compareTo(w)<0;
  }

      private static void exch(Comparable[] a,int i ,int j){
      Comparable t = a[i];
      a[i] = a[j];
      a[j] = t;
  }

  public static void sort(Comparable[] a){
      StdRandom.shuffle(a);
      sort(a,0,a.length-1);
  }

  private static void sort(Comparable[] a,int lo ,int hi){
      if(hi<=lo) return;
      int j = partition(a,lo,hi);
      sort(a,lo,j-1);
      sort(a,j+1,hi);
  }

  private static partition(Comparable[] a,int lo, int hi)
  {
      int i = lo ,j = hi+1;
      Comparable v = a[lo];
      while(true)
      {

          while(less(v,a[--j])) if(j == lo) break;

      exch(a,lo,j);//j放到正确的位置
      return j;

归并排序

归并排序辅助数组只有原数组一半大的归并排序

归并排序的思想是通过将两个有序的序列合并成一个有序的序列。
当left[i]>right[j] origin[k] = right[j]; j++;
当一边取完就从另一边取
将一个无序的数组通过迭代的方法分至最小，即由一个元素构成的序列进行排序即自顶而下的归并排序

import edu.princeton.cs.algs4.StdOut;
public class Merge{
  private static Comparable[] aux;

  @SuppressWarnings("unchecked")
  private static boolean less(Comparable v, Comparable w){
    return v.compareTo(w)<0;
  }

  public static void sort(Comparable[] a){
    aux = new Comparable[a.length];
    sort(a,0,a.length-1);
  }

  private static void sort(Comparable[] a,int lo ,int hi){
    if(hi<=lo)return;
    int mid = lo+(hi-lo)/2;
    sort(a,lo,mid);
    sort(a,mid+1,hi);
    merge(a,lo,mid,hi);
  }

  public static void merge(Comparable[] a,int lo,int mid, int hi){
    int i = lo,j = mid+1;

    for(int k = lo;k<=hi;k++){
        aux[k] = a[k];
    }

    for(int k = lo;k<=hi;k++){
        if(i>mid) a[k] = a[j++];
        else if(j>hi) a[k] = aux[i++];
        else if(less(aux[i],aux[j])) a[k] = aux[i++];
        else a[k] = aux[j++];
    }   
  }

  public static void main(String[] args){
    Comparable[] a = {2,4,6,8,0,1,3,7};
    Merge m = new Merge();
    m.sort(a);
    for(Comparable c :a){
        StdOut.println(c);
    }
  }
}

以插入排序为基础，将每一个对象与其前h个对象进行比较（h为一个递增序列通常为3h+1,比较进行一轮后h/3），当h=1时相当于进行插入排序。原因是插入排序对h有序（对象间隔h个有序）的队列排序时速度较快。

public class Shell{

                    exch(a,j,j-h);
                }
            }
            h = h/3;
        }

import edu.princeton.cs.algs4.LinkedStack;
import java.util.Iterator;
import edu.princeton.cs.algs4.StdOut;
public class Deque<Item> implements Iterable<Item>{
  private LinkedStack<Item> headStack;
  private LinkedStack<Item> endStack;  
  //维护两个栈一个负责头部的加入和移除一个负责尾部的

  public Deque(){
      headStack = new LinkedStack<Item>();
      endStack = new LinkedStack<Item>();
  }

  public boolean isEmpty(){
      return headStack.isEmpty()&&endStack.isEmpty();
      //如果两个栈都为空的话则整个栈为空
  }

  public int size(){
      return headStack.size()+endStack.size();
      //两个栈的size相加等于整个栈
  }

  public void addFirst(Item item){
      if(item != null){
          headStack.push(item);
      }
      else{
          throw new java.lang.IllegalArgumentException();
      }
      //在栈的最前端加入元素等于在头部栈的栈顶加入元素，头部栈的栈顶是整个栈的最前端，尾部栈的栈顶是整个栈的最末端
  }

  public void addLast(Item item){
      if(item != null){
           endStack.push(item);
      }
      else{
          throw new java.lang.IllegalArgumentException();
      }       
  }

  public Item removeFirst(){
      if(!isEmpty()){                    
        //如果头部栈为空的话，要移除第一个元素，需要将尾部栈的元素倒序复制到头部栈
          if(headStack.isEmpty()){
              while(!endStack.isEmpty()){
                  headStack.push(endStack.pop());
              }
          }
          return headStack.pop();
      }
      else throw new java.util.NoSuchElementException();

  }

  public Item removeLast(){
      if(!isEmpty()){         
        //如果尾部栈为空的话，要移除最后一个元素，需要将头部栈的元素倒序复制到尾部栈           
          if(endStack.isEmpty()){
              while(!headStack.isEmpty()){
                  endStack.push(headStack.pop());
              }
          }
          return headStack.pop();
      }
      else throw new java.util.NoSuchElementException();        
  }

  public Iterator<Item> iterator(){
      return new DequeIterator();
  }

  private class DequeIterator implements Iterator<Item>{
      private int i = size();

      Iterator<Item> head = headStack.iterator();
      Iterator<Item> end = endStack.iterator();

      public boolean hasNext(){return i>0;}
      public Item next(){
          if(!hasNext()){throw new java.util.NoSuchElementException();}
          if(!headStack.isEmpty()){
              i--;
              return head.next();
          }
          else if (headStack.isEmpty()){
              while(!endStack.isEmpty()){
                  headStack.push(endStack.pop());
              }
              i--;
              return head.next();
          }  
          else throw new java.util.NoSuchElementException();
      }
      public void remove(){
          throw new java.lang.UnsupportedOperationException();
      }
  }
  //测试函数
  public static void main(String[] args){
      Deque<Integer> dq = new Deque<Integer>();
      dq.addFirst(1);
      dq.addLast(2);
      dq.addFirst(4);
      StdOut.println(dq.removeFirst());
      StdOut.println(dq.removeFirst());
      StdOut.println(dq.removeFirst());
      StdOut.println(dq.size());
  }
}

import java.util.Iterator;
import edu.princeton.cs.algs4.StdRandom;
import edu.princeton.cs.algs4.StdOut;
public class RandomizedQueue<T> implements Iterable<T>{

  private T[] s;//用数组实现队列
  private int N;//当前数组中元素个数

  @SuppressWarnings("unchecked")
  public RandomizedQueue(){
    s = (T[])new Comparable<?>[1];
    N = 0;
  }

  @SuppressWarnings("unchecked")
  private void resize(int capacity){
    //数组大小重置为capacity
    T[] copy = (T[])new Comparable<?>[capacity];
    for(int i = 0;i<N;i++){
        copy[i]=s[i];
    }
    s = copy;
  }    

  public boolean isEmpty(){
    return N == 0;
  }

  public int size(){
    return N;
  }

  public void enqueue(T item){
    if(item ==null){
        throw new java.lang.IllegalArgumentException();
    }
    if(N == s.length){
      //如果数组满了，更改数组大小为当前两倍
        resize(2*s.length);
    }
    s[N++] = item;      
  }

  public T dequeue(){
    if(isEmpty()){
        throw new java.util.NoSuchElementException();
    }        
    int index = StdRandom.uniform(N);
    T temp = s[index];
    s[index] = s[N-1];
    s[N-1] = temp;
    //交换随机索引和最后一个索引对应的值
    T item = s[--N];
    s[N] = null;
    if(N>0&&N<s.length/4){
      //如果数组不满1/4，更改数组大小为当前的1/2
        resize(s.length/2);
    }
    return item;
  }

  public T sample(){
    int index = StdRandom.uniform(N);
    return s[index];
  }

  public Iterator<T> iterator(){
    return new RandomIterator();
  }

  private class RandomIterator implements Iterator<T>{
    private int num =N;     

    public boolean hasNext(){
        return N==0;
    }

    public T next(){
        if(num>0){
            int index = StdRandom.uniform(num--);
            T temp = s[index];
            s[index] = s[num];
            s[num]= temp;
            return temp;      
        }
        else{
            throw new java.util.NoSuchElementException();
        }
    }

    public void remove(){
        throw new java.lang.UnsupportedOperationException();
    }
  }

  public static void main(String[] args){
    RandomizedQueue<Integer> rq = new RandomizedQueue<Integer>();
    rq.enqueue(1);
    rq.enqueue(45);
    rq.enqueue(23);
    rq.enqueue(23214);
    rq.enqueue(43);
    rq.enqueue(123);
    rq.enqueue(435);
    rq.enqueue(234);
    /*
    StdOut.println(rq.sample());
    StdOut.println(rq.dequeue());
    */
    Iterator<Integer> i = rq.iterator();
    while(!i.hasNext()){
        int s = i.next();
        StdOut.println(s);
    }
  }
}

可以返回最大值的栈结构

构造一个数据结构拥有栈的基本操作和一个可以返回最大值的方法

public static class MaxStack{
  private Node first;
  private Node max;

  private class Node{
    double item;
    Node next;
  }

  public MaxStack{
    first = null;
    max = null;
  }

  public double GetMax(){
    return max.item;
  }

  public void push(double item){
    if(item >= max.item){
      oldmax = max.item;
      max = new Node();
      max.item = item;
      max.next = oldmax;
    }
    oldfirst = first.item;
    first = new Node();
    first.item = item;
    first.next = oldfirst;
  }

  public double pop(){
    double temp = first.item;
    first = first.next;
    if(temp == max.item){
      max = max.next;
    }
    return temp;
  }
}

构造一个由两个栈组成的队列

如何构造一个由两个栈组成的队列

队列所需的操作

  • enqueue ：将一个元素加入到队列的末尾
  • dequeue ： 将一个元素从队列的顶部移除

栈的操作

  • pop : 将一个元素从栈顶移除
  • push ：将一个元素加入到栈顶

思路

  • 可以将队列分为两部分 inbox outbox(两个栈)
  • 当outbox为空的时候，将inbox的元素逐个pop并push到outbox中，此时outbox的栈顶为最先加入到inbox的元素

代码

public static class QueueWithTwoStakcs{
  private Stack<Item> inbox = new Stack<Item>;
  private Stack<Item> outbox = new Stack<Item>;

  public void enqueue(Item item){
    inbox.push(item);
  }

  public Item dequeue(){
    if(outbox.isEmpty()){
      while(!inbox.isEmpty){
        outbox.push(inbox.pop());
      }
    }
    return outbox.pop();
  }
}
