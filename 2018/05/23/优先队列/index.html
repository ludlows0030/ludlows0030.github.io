优先队列

优先队列

为什么我们需要优先队列？

很多情况下，我们处理元素时，不一定要求它们全部有序，或是不一定要一次就将它们排序。很多情况下我们会收集一些元素，处理当前键值最大的元素，然后再收集更多的元素，再处理当前键值最大的元素。

例如，可能有一台能够同时运行多个应用程序的电脑。通过为每个应用程序的事件分配一个优先级，并总是处理下一个优先级最高的事件来实现的。

优先队列应该支持什么样的操作

  • 删除最大的元素
  • 插入元素

基于二叉堆数据结构实现的数据队列

API

public class MaxPQ<Key extends Comparable<Key>>
{
  MaxPQ()//创建一个优先队列
  MaxPQ(int max)//创建一个初始容量为max的优先队列
  MaxPQ(Key[] a)//用a[]中的元素创建一个优先队列

  void insert(Key v) //向优先队列中插入一个元素
  Key max() //返回最大元素
  Key delMax() //删除并返回最大元素
  boolean isEmpty() //返回队列是否为空
  int size() //返回优先队列中的元素个数
}

一个优先队列的用例

public class TopM
{
  public static main(String[] args)
  {
    int M = Integer.parseInt(args[0]);
    MinPQ<Transaction> pq = new MinPQ<Transaction>(M+1);
    while(StdIn.hasNextLine())
    {
      pq.insert(new Transaction(StdIn.readLine()));
      if(pq.size()>M)
      {
        pq.delMin();//当元素个数超出时，删除最小的以保证数组内元素为最大的M个
      }
    }

    Stack<Transaction> stack = new Stack<Transaction>();
    while(!pq.isEmpty()) stack.push(pq.delMin());//依次删除最小元素push进栈，栈内为从小到大的顺序
    for(Transaction t : stack) StdOut.println(t);
  }
}

堆的定义

堆有序： 当一棵二叉树的每个结点都大于等于它的两个子节点时，称为堆有序

从任意结点向上，我们都能得到一列非递减的元素；
从任意结点向下，我们都能得到一列非递增的元素。

二叉堆： 一组能够用堆有序的完全二叉树排列的元素，并在数组中按照层级储存（为方便计算，不是用数组的第一个位置）

在一个堆中，位置k的结点的父结点的位置为 k/2，而它的两个子结点的位置分别为2k和2k+1。

堆的算法

为实现优先队列删除最大元素和插入元素的操作，必须先实现堆有序化中，上浮(swim)和下潜(sink)的操作

private void swim(int k)
{
  while(k>1&&less(k/2,k))
  {
    exch(k/2,k);
    k/2;
  }
}

private void sink(int k)
{
  while(k*2<=N)
  {
    int j = k*2;
    if(j<N&&less(j,j+1)) j++;
    if(!less(k,j)) break;
    exch(k,j);
    k=j;
  }
}

  • 插入元素操作

    • 将新元素添加到末尾
    • 使其上浮到合适位置

  • 删除最大元素操作

    • 将根结点和最后一个结点交换
    • 令交换后的根结点下潜到合适位置

基于堆的优先队列

public class MaxPQ<Key extends Comparable<Key>>
{
    private Key[] pq;
    private int N = 0;

    public MaxPQ(int maxN)
    {
        pq = (Key[]) new Comparable[maxN+1];
    }

    public boolean Empty()
    {
        return N == 0;
    }

    public int size()
    {
        return N;
    }

    public void insert(Key v)
    {
        pq[++N] = v;//数组最后添加一个结点
        swim(v);//上浮该结点
    }

    public Key delMax()
    {
        Key max = pq[1];//根节点为最大的
        exch(1,N--);//将根节点和最后一个结点交换
        pq[N+1] = null;//交换后=null防止对象游离
        sink(1);//将交换上来的最后一个结点下沉
        return max;
    }

    private boolean less(int i, int j)
    {
        return pq[i].compareto(pq[j]) < 0;
    }

    private void exch(int i, int j)
    {
        Key temp = pq[i];
        pq[i] = pq[j];
        pq[j] = temp;
    }

    private void swim(int k)
    {
        while(k>1&&less(k/2,k))//当结点到达根节点停止循环或者父节点比当前结点大
        {
            exch(k/2,k);
            k/2;
        }
    }

    private void sink(int k)
    {
        while(k*2<=N)
        {
            int j = k*2;
            if(j<N&&less(j,j+1)) j++;
            if(!less(k,j)) break;
            exch(k,j);
            k = j;
        }
    }
}
