简易3d游戏制作笔记-10

利用Messenger系统传递消息

基本实现思路

  • 复制Wiki上的Messenger到项目工程
  • 一个管理事件清单的GameEvent类
  • 一个广播事件的方法，放在方法需要被广播的地方（例如血量修改后，需要通知其他脚本例如更新UI）

  • 一个接收事件的方法，用于接收广播的消息

  • GameEvent脚本

public static class GameEvent{
  public const string 消息名 = "消息内容";
}
  • 广播方法
Messenger<若有传递的参数可在此声明类型>.Broadcast(GameEvent.消息名,传递的参数);
  • 接收和移除接收的方法
Messenger.AddListener(GameEvent.消息名,接收到消息后调用的方法());

Messenger.RemoveListener(GameEvent.消息名，接收到消息后调用的方法);
