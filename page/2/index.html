修改地面检测处理边缘和斜坡

基本实现思路

  • 改用光线投射检测代替碰撞器的isGrounded检测

  • 当角色模型垂直速度小于0，并且向下的光线投射碰到物体（几乎是所有情况下），如果hit.distance小于一个临界值（check），则判定角色模型位于地面上。此时和前面isGrounded为true时做相同处理。如果判定角色模型不在地面上（hit.distance>check）,并且此时isGrounded为true，则说明角色位于边缘。此时如果角色的movement向量和碰撞点的法向量方向相同，则movement+=normalspeed，否则movement = normalspeed

代码(C#)

private ControllerColliderHit _contact;
//用于储存碰撞数据

实现跳跃动作

角色模型时刻受到重力向下拉，当“跳跃”时，应用向上的力

基本思路

  • 进行一系列的判断，来决定角色模型上收到的vertSpeed为多少。

  • 如果角色模型在地面上时，若按下“跳跃键”，则vertSpeed为一个向上的跳跃力，否则vertSpeed为一个很小的向下的力（保证模型在凹凸不平的地面上行走时，被按在地上）。

  • 当角色模型不在地面上时，vertSpeed+=重力乘以一个系数，并且当vertSpeed大于设定的下落的最大速度时，使其等于该速度。从而避免vertSpeed过大

代码（C#）

void Update(){
  float vertSpeed = minFallSpeed;
  if(charController.isGournded)
  {
    if(Input.GetButtonDown("Jump"))
    {
      vertSpeed = jumpSpeed;
    }
    else
    {
    vertSpeed = minFallSpeed;
    }
  }
  else
  {
    vertSpeed += grivaty*5*Time.deltaTime;
    if(vertSpeed<maxFallSpeed)
    {
      //注意此处的判断为vertSpeed<maxFallSpeed,因为两者都为负数
      vertSpeed = maxFallSpeed;
    }
  }
  movement.y = vertSpeed;
}

Camera-relative移动

基本实现思路

  1. 先实现角色模型的旋转
    1.1 获取按键的水平和垂直输入并保存在一个零向量中
    1.2 使角色围绕摄像机的y轴进行旋转
    1.3 插值运算使旋转平滑

    代码（C#）

    void Update(){
Vector3 movement = new Vector3.zero;

float horInput = Get.Axis("Horizontal");
float vertInput = Get.Axis("Vertical");
if(horInput!=0&&vertInput!=0){
  movement.x = horInput;
  movement.z = vertInput;
}

Quaternion temp = target.rotation;
target.eulerAngles = new Vector3(0,target.eulerAngles.y,0);
movement = target.TransformDirection(movement);
target.rotation = temp;

Quaternion direction = Quaternion.LookRotation(movement);
transform.rotation = Quaternion.Lerp(transform.rotation,direction,rotSpeed*Time.deltime);
}

  2. 实现角色的移动
    2.1 给角色模型添加CharacterController组件
    2.2 给水平和垂直输入乘以一个移动的速度
    2.3 限制对角线移动时的速度

    代码（C#） //以前面的代码为基础

    void Start(){
  charController = GetComponent<CharacterController>();
}

void Update(){
Vector3 movement = new Vector3.zero;

float horInput = Get.Axis("Horizontal");
float vertInput = Get.Axis("Vertical");
if(horInput!=0&&vertInput!=0){
  movement.x = horInput*moveSpeed;
  movement.z = vertInput*moveSpeed;

  movement = Vector3.ClapMagnitude(movement,moveSpeed);
}

Quaternion temp = target.rotation; //target为摄像机Transform
target.eulerAngles = new Vector3(0,target.eulerAngles.y,0);
movement = target.TransformDirection(movement);
target.rotation = temp;

Quaternion direction = Quaternion.LookRotation(movement);
transform.rotation = Quaternion.Lerp(transform.rotation,direction,rotSpeed*Time.deltime);

movement*=Time.deltime;
charController.Move(movement);
}

发表于 2018-03-29

摄像机环绕角色进行旋转

基本实现思路

  1. 获得摄像机和角色模型的初始Transform偏移值
  2. 将按键和鼠标水平移动得到的水平输入值加到摄像机的旋转的y值上，获得旋转后的摄像机的Rotation
  3. 通过角色模型的位置减去旋转后的偏移值，得到摄像机的当前应该处于的位置
  4. 通过代码使摄像机面对模型

代码（C#）：

void Start(){
  _rotY = target.eulerAngles.y;
  _offset = target.position-transform.position;
  //target为角色模型的Transform
}

void LateUpdate(){
  float horInput = Input.GetAxis("Horizontal")
  if(horInput!=0){
    _rotY += horInput*rotSpeed;
  }
  else{
          _rotY += GetAxis("Mouse X")*rotSpeed*3
          //鼠标的水平移动比键盘的更灵敏一些
  }
  Quaternion rotation = Quaternion.Euler(0,_rotY,0);
  //将欧拉角转化为四元数
  transform.position = target.position-(_offset*rotation);
  //Vector3乘以一个四元数得到旋转后的Vector3
  transform.LookAt(target);
}
